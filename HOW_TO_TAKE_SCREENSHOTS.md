## üõ†Ô∏è Step-by-Step Guide

### Using Chrome DevTools (Easiest):

1. **Open your app** in Chrome
2. **Press F12** (or Cmd+Option+I on Mac) to open DevTools
3. **Click the device icon** (üì±) in the top-left of DevTools
4. **Set dimensions:**
   - Click "Responsive" dropdown
   - Select "Edit..."
   - Enter: Width: `1280`, Height: `720`
   - Press Enter
5. **Refresh the page** (F5) to see it at that size
6. **Take screenshot:**
   - **Mac:** Cmd+Shift+4, then drag to select area
   - **Windows:** Use Snipping Tool or Win+Shift+S
   - **Or:** In DevTools, click the "..." menu ‚Üí "Capture screenshot"

### Using Browser Extensions:

**Recommended:** [Full Page Screen Capture](https://chrome.google.com/webstore/detail/full-page-screen-capture/fdpohaocaechififmbbbbbknoalclacl)

1. Install extension
2. Set viewport to 1280x720
3. Click extension icon
4. Save screenshot

## üé® Pro Tips

1. **Use real data** - Connect a test Shopify store to show actual data
2. **Show value** - Make sure screenshots highlight what makes your app useful
3. **Consistent styling** - All screenshots should have the same viewport size
4. **Clean UI** - Make sure no browser toolbars or bookmarks are visible
5. **Good lighting** - Use light mode (your app already has good contrast)

## üöÄ Quick Method (5 minutes)

**Fastest way to get all screenshots:**

1. Open Chrome DevTools (F12)
2. Set viewport to 1280x720
3. Go through your app:
   - Dashboard ‚Üí Screenshot

   - Click "Generate Report" ‚Üí Wait ‚Üí Screenshot
   - Go to Settings ‚Üí Screenshot
4. Done! You have 5 screenshots
