### 2. Screencast URL

**This field is REQUIRED** - You need to provide a video URL.

#### Option A: Create a Screencast (Recommended)

**What to include in the video (3-8 minutes):**

1. **Onboarding Flow (1-2 min):**
   - Show registration page
   - Create new account
   - Login process
   - Dashboard overview

2. **Core Features (2-5 min):**
   - **Order Processing:**
     - Navigate to order processing
     - Show pending/unfulfilled orders
     - Explain what the feature does
   
   - **Inventory Management:**
     - Show inventory dashboard
     - Demonstrate low-stock alerts
     - Show search/filter functionality
     - Show CSV export
   
   - **Revenue Analytics:**
     - Show revenue dashboard
     - Display product breakdown
     - Show top products
     - Show CSV export

3. **Shopify Connection (1 min):**
   - Go to Settings
   - Show "Connect Shopify Store" section
   - Explain OAuth flow (or manual connection)

4. **Subscription (30 sec):**
   - Show subscription page
   - Explain pricing ($29/month)
   - Mention 7-day free trial

**Where to host the video:**
- **YouTube** (Recommended - Free, reliable)
  - Upload as "Unlisted" (not public, but accessible via link)

  - Upload as "Unlisted"
  - Copy the video URL

- **Loom** (Quick option)
  - Record screen directly
  - Get shareable link

### Step 3: Create Screencast Video

**Quick recording guide:**

1. **Prepare your environment:**
   - Have a test account ready
   - Have a Shopify development store connected (optional but helpful)
   - Clear browser cache for clean look
   - Use a clean browser window

2. **Record using Loom (easiest):**
   - Go to https://loom.com
   - Sign up (free)
   - Click "New Recording"
   - Select "Screen + Camera" or "Screen Only"
   - Record your screen showing:
     - Registration ‚Üí Login ‚Üí Dashboard
     - Order Processing feature
     - Inventory Management feature
     - Revenue Analytics feature
     - Settings/Shopify connection
   - Stop recording
   - Copy the shareable link

3. **Or use YouTube:**
   - Record screen using QuickTime (Mac) or OBS
   - Upload to YouTube
   - Set visibility to "Unlisted"
   - Copy video URL

---

### Step 4: Fill in Screencast URL

1. In the "Screencast URL" field
2. Paste your video URL (YouTube, Vimeo, or Loom)
3. Verify the URL works by clicking it
